## Installation
Edit your build.gradle file and add below dependency:
```gradle
    dependencies {
        implementation 'org.forgerock.android:forgerock-auth:1.0.00'
        implementation 'org.forgerock.android:forgerock-auth-ui:1.0.00' // (Optional) Sample UI
    }
```
